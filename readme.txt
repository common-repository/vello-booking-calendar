Plugin to Show Vello Booking Calendar with Shortcode.

== Description ==
This plugin provides shortcode [vello] for easy booking calendar integration on a Wordpress site.

Vello Booking Calendar requires an active Vello account. Create and manage your booking calendar at [www.vello.fi/en](https://vello.fi).

== Installation ==
Activate plugin and insert Vello id in booking calendar menu.
